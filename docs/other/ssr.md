# ShadowsocksR
- 前提条件

系统支持：CentOS 6+ / Debian 6+ / Ubuntu 14+
<br>语言支持：Python2

- 安装脚本
```shell
wget -N –no-check-certificate https://github.com/slinjing/script/blob/main/ssr.sh && chmod +x ssr.sh && bash ssr.sh
```
此脚本是开机自动启动，部署一次即可，部署完成后需要在安全组放行对于端口。
